# Тестовое задание: Загрузка данных из API

## Описание
Проект загружает данные из внешнего API (продажи, заказы, склады, доходы) и сохраняет их в JSON файлы.

## Используемые технологии
- PHP 8.1
- GitHub Pages

## Эндпоинты
- Продажи: /api/sales
- Заказы: /api/orders
- Склады: /api/stocks
- Доходы: /api/incomes

## Как запустить
1. Открыть index.php в браузере (через GitHub Pages)
2. Данные автоматически загрузятся и сохранятся в JSON файлы

## Результат
- sales.json - данные по продажам
- orders.json - данные по заказам
- stocks.json - данные по складам
- incomes.json - данные по доходам
